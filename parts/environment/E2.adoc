=== (E.2) Components

The system will use different components from the environment and will provide components to the user to interact with the system. 

*Local File System*: Stores fragment images and the segmented image cache used by the desktop app.

*SQLite Database*: Stores fragment metadata, custom filters, and project states.

*ML Pipeline Command-Line Interface (CLI)*: A local interface used to run segmentation and metadata extraction processors.
