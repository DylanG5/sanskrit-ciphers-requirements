==== Overview

The system will provide several interfaces to facilitate interaction between users, the local data store, and the offline ML pipeline.

*User Interface*: The system will provide a desktop (Electron) user interface for project management, fragment search, filtering, canvas manipulation, metadata editing, and uploads.

*Local IPC Interface*: The system will expose a local IPC API between the renderer UI and the Electron main process for fragments, projects, files, and custom filters.

*ML Pipeline CLI Interface*: The system will provide a command-line interface for running the ML pipeline (list, migrate, run, status) to enrich fragment metadata.

*File System Interface*: The system will use a local data directory for images and a cache directory for segmented PNGs.
