=== (S.1) Components

The system consists of three main clusters that work together to support local fragment exploration, reconstruction, and ML-assisted metadata enrichment.

==== Front-end Cluster

* *Home & Search Interface*: Project list and fragment ID search with autocomplete.
* *Fragment Browser & Filters*: Sidebar list with pagination and filters for line count, script type, edge flags, circle detection, and custom fields.
* *Interactive Canvas Module*: Drag-and-drop placement with move/resize/rotate, multi-select, lock/unlock, grid toggle, and segmented/original view toggle.
* *Metadata Editor*: View and edit fragment metadata (line count, script type, edge flags, circle, scale, custom fields).
* *Upload Dialog*: Import new fragment images into the local data store.
* *Project/Session Management*: Create, rename, delete, save, and restore canvas projects with auto-save.

==== Backend Cluster

* *Electron IPC Layer*: Local interface between the UI and backend services.
* *Fragment Data Service*: Query, search, and update fragment metadata.
* *Project Persistence Service*: Save/load canvas state.
* *File Management Service*: Image import, storage, and segmented image cache.
* *Edge Matching Service*: Backend edge comparison for candidate fragment matches.

===== Data Storage Cluster

* *Fragment Database (SQLite)*: Image paths and metadata (edges, line counts, script type, segmentation, scale, circle).
* *Project Database (SQLite)*: Project records and stored canvas layouts.
* *Custom Filter Registry*: User-defined metadata fields and options.

==== AI/ML Cluster

* *Segmentation Processor (You Only Look Once, YOLO)*: Extracts contours and produces transparent fragments for the UI.
* *Line Count Classifier (Convolutional Neural Network, CNN)*: Predicts number of text lines.
* *Line Detection Processor (YOLO)*: Detects individual text lines and stores line boxes/count.
* *Script Type Classifier*: Predicts script type labels with confidence.
* *Edge Detection Processor*: Infers border edges and edge-piece flags from contours.
* *Circle Detection Processor*: Detects circular marks.
* *Scale Detection Processor*: Estimates pixels-per-unit from rulers using optical character recognition (OCR) and computer vision (CV).
